API Endpoints

| Method | Endpoint        | Description       |
| ------ | --------------- | ----------------- |
| GET    | /api/posts      | List all posts    |
| GET    | /api/posts/{id} | Get a post by ID  |
| POST   | /api/posts      | Create a new post |
| PUT    | /api/posts/{id} | Update a post     |
| DELETE | /api/posts/{id} | Delete a post     |
